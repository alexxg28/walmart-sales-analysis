# walmart-sales-analysis
Data analysis of Walmart's weekly sales data across 45 stores from 2010 to 2012, using Python (Pandas, Seaborn, Matplotlib).


Questions Explored

Which stores are most sensitive to temperature changes?
Which stores were the top 5 highest in revenue from 2010 to 2012?
Does unemployment influence overall store revenue?


Key Findings

Top-performing stores averaged 3.5 points lower unemployment than the lowest-performing stores, with a $23M+ revenue gap between the two groups
High-revenue stores tended to operate in milder temperatures (~60°F) with lower unemployment rates
Stores 43 and 44 showed the highest sensitivity to temperature changes


Tools Used

Python, Pandas, Seaborn, Matplotlib, Google Colab
